MosquitoReportAPI

This API exposes endpoints to search and report areas where mosquitoes could spread. Areas of stagnant water, such as flower vases, uncovered barrels, buckets, and discarded tires, can be reported. The main goal is to support the combat against diseases like Dengue, Zika and Chikungunya.

$ curl https://api.mosquito.report/stats
{
  "reports": {
    "week": 0,
    "month": 0,
    "ever": 0

  1. Overview

    1. Authentication

      In order to interact with non public endpoints, the client must provide an authentication token. The authentication token can be obteined using the following endpoints:

      • Authenticate an user using his email
      • Authenticate an user using a Facebook token

      The authentication token must be provided via the "Authorization" http header. Example: 

      Authorization: Token ZXhhbXBsZXRva2VuNjI4NzMzMWMtYWIwZi00ODMxLWFhMjQtNWUyYzYyZmM2MmYwDQo=

      If an invalid token was provided then the API will return the following error:

      HTTP/1.1 403 Forbidden 
      {
  "status":"error",
  "message":"Invalid authorization token"
}

    2. Validation

      Response for invalid json:

      HTTP/1.1 400 Bad Request 
      {
  "status":"error",
  "message":"Invalid json"
}

      Response for invalid json parameters:

      HTTP/1.1 400 Bad Request 
      {
  "status":"error",
  "message":"Error mapping json"
}
